# Team_X_IP_OSINT
Получение информации о IP-адресе цели
Скрипт написан на основе ipapi.co

Для установки:
$ git clone https://github.com/teamXofficial/Team_X_IP_OSINT
$ cd Team_X_IP_OSINT
$ chmod +x *
Запуск:
$ python3 main.py
После введите IP-адрес цели и вы получите много информации о ней.
